## Get Started

```bash
git clone https://github.com/ogdenjs/callbacks-promises.git
cd callbacks-promises
npm install

# to run examples #
node examples/1-say-hello.js

# to run examples written in ES2015 #
./node_modules/.bin/babel-node examples/4-async-await.js
```
